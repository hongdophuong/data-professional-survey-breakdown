Data Professional Survey Breakdown 

1- Project Description: 
This project analyzes a survey of data professionals to explore salary trends, programming language preferences, career entry difficulty, and job satisfaction across different roles in the data industry.
Using Power BI, the dashboard transforms raw survey data into an interactive report that highlights key insights about the data career landscape, including salary differences between roles, popular technical skills, and satisfaction with work-life balance.
The goal of the dashboard is to provide a clear overview of the data profession ecosystem and help stakeholders quickly understand trends in the data job market.

Tools Used
- Power BI
- Data Cleaning & Transformation (Power Query)
- Data Modeling
- Data Visualization & Dashboard Design

2- Key Insights: 
- Data Scientists earn the highest average salary (~88k USD)
- Python is the most preferred programming language among data professionals
- The majority of respondents find entering the data field moderately difficult
- Professionals report higher satisfaction with work-life balance than salary
- The United States contributes the largest share of survey respondents
